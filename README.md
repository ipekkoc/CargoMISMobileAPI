# Cargo MIS API

Die Cargo MIS API ist eine RESTful Backend-Anwendung zur Verwaltung
logistischer Geschäftsprozesse.  
Sie stellt strukturierte Schnittstellen zur Verarbeitung von Aufträgen,
Sendungen und zugehörigen Statusinformationen bereit.

Der Fokus des Projekts liegt auf einer klaren Backend-Architektur,
REST-konformen API-Endpunkten und einer übersichtlichen Projektstruktur.


## Projektbeschreibung
Die Anwendung bildet zentrale Abläufe eines logistischen
Management-Informationssystems ab.  
Dazu gehören die Erfassung, Verwaltung und Abfrage von Aufträgen sowie
die Verwaltung von Sendungen und deren Status.

Das System ist modular aufgebaut und ermöglicht eine einfache Erweiterung
um zusätzliche Funktionen oder Module.


## Funktionalitäten
- Verwaltung von Aufträgen
- Verwaltung von Sendungen und Statusinformationen
- REST-konforme API-Endpunkte
- Modularer und erweiterbarer Aufbau


## Technologien
- Node.js
- Express.js
- JavaScript
- REST API
- Git & GitHub
